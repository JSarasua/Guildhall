How to Use:
Starting Visual Unit Tests shows a Capsule, AABB2, Disc, OBB2, and Line Segment randomly in space. 
	Keyboard controls:
		Esc: Closes application
		R:	Regenerates positions of shapes
		T:	Toggles Point versus AABB at mouse cursor
		Y:	Rotates AABB at mouse cursor
